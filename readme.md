# gausian copula
 a Gaussian Copula is a mathematical function that allows you to model the relationships (correlations) between multiple variables completely separately from the individual shapes (distributions) of those variables
 The entire process is anchored by Sklar's Theorem, which states that any complex multivariate distribution can be broken down into its individual (marginal) distributions and a "copula" that mathematically links them together.
